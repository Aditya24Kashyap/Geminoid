# Geminoid – Gemini-like AI Assistant Web App

**Geminoid** is an interactive AI assistant web app inspired by Google's Gemini, built with **React**, **Vite**, **Tailwind CSS**, and **Zustand**. It uses the **Gemini API** (or your chosen LLM API) to generate intelligent responses in real-time, mimicking modern AI assistants like ChatGPT or Gemini.

---

## 🚀 Features

- 🔮 Chat interface with Gemini-style UI
- 💬 Real-time AI responses using Gemini API
- 📜 Prompt history with automatic scrolling
- 🧠 Global state management via Zustand
- ⚡ Fast development with Vite
- 🎨 Clean, responsive design with Tailwind CSS
